#### print("{:n}".format(a)) 控制输出的长度
``` python
print("{:5}".fromat(int(i*j)),end=" ")
```

### label
a=[1,2,"a","c"]
#### 类型无关
#### 通过索引查找
``` python
>>> a[1]
2
>>> a[-1]
c
#负数索引从右边开始计数，并且从一开始而非从零开始
```
#### 索引与切片操作
```python 
>>> a[1:-1]
[2,'a','c']
>>> a[:]
[1,2,'a','c']
#省略的索引默认填充为最左/最右内容索引值
#当索引左界限过大返回空列表 右界限过大返回左界限后到结束的全部内容
>>> a[1::2]
[2,'c']
#可以设定步长 上行为2（2个里取一个）
```

#### 修改索引内容
可以使用= +直接修改
##### 利用切片清空索引
```python 
>>>a[:]=[]
>>>a
[]
```
